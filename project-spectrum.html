Lorentzian Spectrum Fitting

Physical system: data from two nearby emission lines from the same species, think a fine-structure doublet in a gas cell. Finite lifetimes and the spectrometerâ€™s slit function broaden each line into a Lorentzian, so the detector sees a blended profile rather than two clean spikes.

Math: model \(S(\nu) = c_1 L(\nu;\nu_{01},\gamma_1) + c_2 L(\nu;\nu_{02},\gamma_2)\) with \(L(\nu;\nu_0,\gamma) = \frac{1}{\pi}\frac{0.5\gamma}{(\nu-\nu_0)^2 + (0.5\gamma)^2}\). Nonlinear least squares (SciPy) tunes amplitudes \(c_1,c_2\), line centers \(\nu_{01},\nu_{02}\), and widths \(\gamma_1,\gamma_2\); the animation walks those six parameters from an initial guess to the fitted values while the residuals collapse.
